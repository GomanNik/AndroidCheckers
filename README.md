# Шашки (Android, Java)

Учебное Android-приложение «Шашки» на Java.  
Проект используется как единая база для выполнения лабораторных работ по Android у Климова: UI, навигация, настройки, уведомления, локализация и т.д.

Приложение полностью **локальное**:
- без выхода в интернет;
- без онлайн-игры;
- без рекламы и сторонней аналитики.

## Основной функционал

- Игра в шашки на одном устройстве:
  - классические правила;
  - игра против ИИ;
  - игра вдвоём на одном устройстве.
- Несколько уровней сложности ИИ:
  - Лёгкий / Средний / Сложный / Эксперт / Гроссмейстер.
- Поддержка разных правил:
  - «Бить обязательно» (можно включить/выключить);
  - подсказка хода (подсветка допустимых ходов).
- Статистика по уровням сложности:
  - количество сыгранных партий;
  - победы / поражения.
- Экран настроек:
  - выбор цвета фигур (играть за белых / чёрных);
  - включение/отключение обязательного боя;
  - включение/отключение подсказки хода;
  - выбор темы (светлая / тёмная);
  - выбор языка интерфейса.
- Экран «Правила игры»:
  - краткое описание правил шашек;
  - условия победы и ничьей;
  - пояснение режимов и настроек (подсказка хода, обязательный бой и т.п.).
- Локальные уведомления:
  - напоминание «Вы давно не играли в шашки» раз в день
    (можно отключить в настройках).

## Экраны приложения

### Главный экран

- Заголовок «Шашки».
- Кнопки:
  - **Играть** — выбор уровня и начало партии против ИИ.
  - **Правила** — переход на экран с описанием правил игры.
  - **Игра вдвоём** — игра на одном устройстве «человек против человека».
  - **Статистика** — просмотр результатов по уровням сложности.
  - **Настройки** — выбор параметров игры и интерфейса.

### Экран выбора уровня сложности

- Список уровней: Лёгкий, Средний, Сложный, Эксперт, Гроссмейстер.
- Выбранный уровень используется для настройки ИИ и статистики.

### Игровое поле

- Доска 8×8, классическая расстановка белых и чёрных шашек.
- Отображение текущего счёта и чьих ход.
- Кнопки управления:
  - выход в главное меню;
  - новая партия / перезапуск;
  - отмена последнего хода (если разрешено правилами);
  - включение/выключение звука.
- Реализация:
  - ход обычных шашек и дамок;
  - взятие шашек и серия взятий;
  - учёт настроек «Бить обязательно» и «Подсказка хода».

### Экран статистики

- Разделён по уровням сложности.
- Для каждого уровня:
  - Сыграно;
  - Побед;
  - Поражений.
- Данные хранятся локально и сохраняются между запусками.

### Экран настроек

- **Играть за:** Белые / Чёрные.
- **Бить обязательно:** Да / Нет.
- **Подсказка хода:** Да / Нет.
- **Цвет фона:** Светлый / Тёмный.
- **Выбор языка:** Русский / English (и другие, при необходимости).

### Экран «Правила игры»

- Локальная страница с правилами шашек:
  - основные ходы;
  - превращение в дамку;
  - правила взятия;
  - условия победы / ничьей;
  - пояснение влияния настроек (обязательное взятие, подсказка и т.п.).

## Цели проекта

- Отработать базовые темы курса Android:
  - разметка экранов (`XML`), кнопки, обработчики событий;
  - несколько `Activity`, передача данных между экранами;
  - темы и стили приложения;
  - работа с ресурсами, строками и локализацией;
  - сохранение настроек через `SharedPreferences`;
  - экраны статистики и списки (через `RecyclerView`, при необходимости);
  - уведомления (`NotificationManager`);
  - `WebView` и работа с локальными HTML-страницами;
  - жизненный цикл Activity и обработка поворота экрана.
- Собрать все лабораторные работы в **один целостный проект**, а не набор разрозненных примеров.

## Технологии

- **Язык:** Java
- **Минимальный SDK:** 24 (Android 7.0, Nougat)
- **IDE:** Android Studio
- **Сборка:** Gradle (Kotlin DSL, `build.gradle.kts`)
- **UI:** классические `View` и `XML`-разметка (без Jetpack Compose)

## Структура проекта

- `app/src/main/java/ru/goman/checkers/`  
  Логика приложения и экраны (`Activity`, вспомогательные классы).

- `app/src/main/res/layout/`  
  XML-разметка экранов (меню, игра, статистика, настройки, правила).

- `app/src/main/res/values*/`  
  Ресурсы строк, цветов, тем (в том числе для разных языков).

- `app/src/main/assets/` *(план)*  
  Локальные HTML-файлы для экрана «Правила игры».

## Сборка и запуск

1. Открыть проект в **Android Studio** (`File → Open…`).
2. Дождаться загрузки зависимостей и завершения Gradle Sync.
3. Подключить Android-устройство или запустить эмулятор.
4. Нажать кнопку **Run** (зелёный ▶) и выбрать устройство.
